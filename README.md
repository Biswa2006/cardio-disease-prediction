# Cardio Disease Prediction System

A complete machine learning pipeline for cardiovascular disease prediction using feature engineering, SMOTE (handling class imbalance), multiple models, and explainable AI (SHAP).

---

## 🚀 Key Features

* Predicts cardiovascular disease risk from patient data
* Handles imbalanced dataset using SMOTE
* Uses multiple ML models (Random Forest, XGBoost, Logistic Regression, SVM)
* Feature selection and preprocessing pipeline
* Model explainability using SHAP
* Modular and scalable project structure

---

## 🧱 Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* XGBoost
* SHAP
* Matplotlib, Seaborn

---

## 📊 Workflow

1. Data preprocessing
2. Feature selection
3. SMOTE balancing
4. Model training
5. Model evaluation
6. Explainability using SHAP
